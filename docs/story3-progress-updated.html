3. Zone2Scene Interactive Elements

  • Branch Tokens: Canopy token in C1, Root token in C2
  • Zone Gates: Visual colliders at B→C1, B→C2, and D entry

4. Gating Logic Flow

  1. Starting state: B→C1/C2 gates RED/LOCKED, D gate RED/LOCKED
  2. Clear 3 marker pads → B→C1/C2 gates turn GREEN/OPEN
  3. Collect canopy token in C1 → state recorded
  4. Collect root token in C2 → state recorded
  5. With both tokens → D gate turns GREEN/OPEN
  6. Feedback messages guide player at each step
