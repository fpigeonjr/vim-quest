⚙ Architecture Notes
// Scene lifecycle: World ↔ Dungeon transition // Enter dungeon: cameras.main.fadeOut(350) → scene.sleep('world') // hides + stops updating world → scene.launch('dungeon') // creates + starts dungeon → dungeon cameras.fadeIn(300) // Exit dungeon (Esc or exit portal): cameras.main.fadeOut(350) → scene.stop('dungeon') // destroys dungeon, frees memory → scene.wake('world') // resumes + shows world → world cameras.fadeIn(350) // triggered by 'wake' event // Guard prevents double-entry during fade: private enteringDungeon = false; if (this.enteringDungeon) return; this.enteringDungeon = true;
