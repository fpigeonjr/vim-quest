# Vim Quest

A browser-based adventure game inspired by Vim Adventures and classic Zelda games. Learn Vim commands by exploring Textland as a blinking cursor hero.

## Quick Start

Open `index.html` in your browser to play.

## Controls

- **Movement**: `h` (left), `j` (down), `k` (up), `l` (right)
- **Word navigation**: `w` (next word), `b` (previous word)
- **Line navigation**: `0` (start of line), `$` (end of line)
- **Actions**: `x` (remove obstacles), `i` (enter insert mode), `Esc` (exit insert mode)

## Game Concept

Navigate through text-based puzzles that teach core Vim motions and commands. Each room contains challenges that require mastering specific Vim shortcuts to progress.
